# Routing Auditor — project instructions

## Git workflow (standing rule)

This repo is kept as a **single squashed commit**. When committing changes here:

1. Squash everything into one commit (amend onto the existing root commit — the
   repo history must stay at exactly one commit).
2. Force-push to `origin main`.

This is the default for every commit in this repo and stays in effect until
explicitly changed. Any change to this rule must be recorded by editing this file.
